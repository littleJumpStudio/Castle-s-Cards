# Castle-s-Cards
Jogue Castle's Cards e experimente o jogo desenvolvido como apoio inteligências artificiais. Com o objetivo de oferecer uma jogabilidade única e progressiva, este jogo de corrida infinita, para dispositivos móveis, apresenta um labirinto de ilusões onde o jovem herói solitário deve encontrar sua musa inspiradora que se encontra encarcerada nas profundezas do submundo.
